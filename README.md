# Tripscripter-front-end
TripScripter - Sistema de Planejamento de Viagens

> Acesse o site online aqui: https://satheuss.github.io/Tripscripter-front-end/

# Sobre o Projeto
O **TripScripter** é um sistema inteligente focado no planejamento e organização de roteiros de viagem. Desenvolvido como parte do Projeto Integrador do curso de Análise e Desenvolvimento de Sistemas (CEUB), o objetivo da plataforma é centralizar e simplificar a criação de itinerários, permitindo que o viajante monte seu dia a dia de forma visual e prática.

# Principais Funcionalidades (Front-end)
* Criação de Roteiros: Organize múltiplas viagens e defina os períodos de cada uma.
* Planejamento Diário: Adicione e gerencie atividades divididas por dias específicos.
* Mapas Interativos: Visualize a localização e os pontos de interesse diretamente na tela do mapa para não se perder.
* Bloco de Notas Integrado: Adicione observações, lembretes e detalhes importantes para cada atividade programada.

# Tecnologias Utilizadas
Para garantir uma aplicação front-end leve, rápida e responsiva, o projeto foi desenvolvido de forma nativa utilizando:
* HTML5 & CSS3 (Estruturação e design moderno alinhado com o protótipo)
* JavaScript (Vanilla) (Lógica do sistema e controle de estado do roteiro)
* Leaflet.js (Integração e renderização de mapas interativos)
